# node-restful-api
RESTful API with Node.js Express

## Development ##
* **Requirements**
    * node: ```v19.1.0``` 
* **Installation**
    * ```npm install```
    * Copy nodemon-example.json and rename it nodemon.json. Edit by adding a valid MONGO_URI
* **CLI Tasks**
    * Start Server: ```npm start```
    * Development Env.: ```npm run dev```
